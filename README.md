# WealthPath Investment Advisory Cloud

A high-fidelity Cloud Engineering simulation project built to showcase scalable web infrastructure deployment on Amazon Web Services (AWS). Developed utilizing Python Flask, an adaptive corporate UI theme, and a securely engineered MySQL transactional database backend framework.

## 🌟 Strategic Core Features
- **Administrative Portal Access**: Secure, single-point authorization node protected against SQL injection vulnerabilities through robust parameterized queries.
- **Dynamic Structural Metrics Dashboard**: Computes asset metrics, total active client arrays, and tracking systems indicators effortlessly.
- **Integrated Client Directory**: Direct form interface mapping registration logic structures tracking legal name identity, metrics, and regional data arrays.
- **Portfolio Management Assets Balancing Ledger**: Flexible transactional manager interface featuring responsive visualization indicator classifications based on variable risk margins.
- **Automated Cloud Provisioning Engine**: Built-in system infrastructure tooling to programmatically process continuous application deployment pipelines, logging checks, and data snapshots.

## 🛠 Architectural Technology Component Suite
- **Backend Core**: Python 3.x via lightweight asynchronous worker engine (Flask micro-framework)
- **Database Engine**: relational MySQL Community Server engine instance layer
- **Interface Layout Matrix**: Structural HTML5, Native CSS Grid Engine Tokens, Bootstrap v5 Framework UI Systems
- **Database Driver Interface**: PyMySQL Native Client Layer Pipeline

## ☁️ AWS Cloud Infrastructure Services Utilized
- **Amazon EC2 (Elastic Compute Cloud)**: Compute capacity layer running via an Ubuntu Server minimal footprint distribution.
- **AWS Security Groups**: Custom network traffic control policies mapping inbound configuration filters across ingress ports (`22` for secure SSH access, `5000` for application evaluation).
